# Raspberry Pi Pico BME280 Sensor with Micropython Library

## About
This is a micropython library for BME280 Combined Humidity and Pressure Sensor

## Setup


I2C pins as shown in the table below:
 
| Raspberry Pi Pico    | Pin | BME280 |
| ---                  | --- | ---    |
| GP16                 | PIN21     | SDA      |
| GP17                 | PIN22     | SCL      |
| GND                  | GND       | GND      |
| Pipe                 | 3.3V(OUT) | VIN      |
